Florida Sinkhole Risk and Vulnerability Multi-Criteria Evaluation Model

This model was created using several data points including: precipitation, geology, and aquifers. These were then crosswalked to for a criteria evaluation scale from 0-10. These data were then given weights in the TerrSet program to determine importance. The final result is the fourth image, bottom left. The last image contains well points, which are important to futher deterioration. This work was completed Fall of 2017 in collaboration with Dr. Jason Evans, Associate Professor of Environmental Science and Studies as Stetson University. You can read my final report here.
